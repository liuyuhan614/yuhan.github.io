# 刘于涵的个人学术主页

这是一个基于HTML、CSS和JavaScript的个人学术主页，展示个人信息、学术成果以及医学影像知识图谱。

## 项目结构

- `index.html` - 主页面文件
- `style.css` - 样式文件
- `script.js` - 交互脚本
- `photo.jpg` - 个人照片（请自行添加）
- `README.md` - 项目说明

## 功能特点

1. **响应式设计** - 适配各种屏幕尺寸
2. **动态导航条** - 固定顶部导航，带有悬停效果
3. **进度条动画** - 学术成绩可视化展示
4. **交互式知识图谱** - 基于D3.js的磁共振成像知识图谱
5. **联系表单** - 用户可发送消息

## 部署说明

1. 将项目文件上传至GitHub仓库
2. 启用GitHub Pages功能
3. 访问 `https://[用户名].github.io/[仓库名]` 查看主页

## 技术栈

- HTML5
- CSS3 (Flexbox, Grid, 动画)
- JavaScript (ES6+)
- D3.js (数据可视化)
